Euler's Method

Initial Value Problem

The initial value problem is as follows:

Given \(\frac{dy}{dx}\) and an initial point \((x_0, y_0\)) find the function \(y\).

These problems are often solved by integrating \(\frac{dy}{dx}\) and plugging in the initial point to solve for the constant.

However, it may be difficult or impossible to integrate the given differential.

Euler's method can help us to approximate the behaviour as well as points of the function.

Derivation

$$m = \frac{\Delta y}{\Delta x}$$ $$\Delta y = m \Delta x$$ $$\Delta y = \frac{dy}{dx} | _{(x_n, y_n)} \Delta x$$ $$y_{n+1} = y_n + \Delta y$$ $$y_{n+1} = y_n + \frac{dy}{dx} \Delta x$$

Given \(x_0\) and \(y_0\) we can approximate many values of the function. A smaller \(\Delta x\) reduces error.

The Method

Since we have a starting point, we can repeatedly find successive points on the function by using the equation described above.

Notice how near the tangent point, the tangent line is really close to the curve. Points in this vicinity are almost on the function. If we repeatedly move rightwards and pick a point on the tangent, we can construct the function from the derivative with the initial point.

Euler's method basically traces a function using the slope field.

Example Problem 1

There is a special horse on a merry-go-round that makes Atharva very happy. As the merry-go-round turns, the horse may go out of view, lowering Atharva's happiness. At \(t\) seconds from the start, Atharva's happiness level changes at a rate of \(e^{\sin \pi t} \sin \pi t\) per second. If Atharva's happiness at \(0\) seconds is \(1\) then:

What is Atharva's happiness after \(10\) seconds?

When is Atharva's happiness \(9\)?

Answer

Atharva's happiness at \(t = 10\) is \(6.65\). Atharva's happiness is \(9\) at \(t = 12.77, 13.69, 14.22\).

We are unable to antiderive \(\frac{dy}{dx} = e^{\sin x} \sin x\), so we will use Euler's Method to graph the antiderivative.

Inaccuracies of Euler's Method

While Euler's method is very useful for approximating the behaviour and values of functions given their derivative and an initial value, the graph generated is not always accurate and may have errors.

Euler's method will not work accurately on functions which have discontinuities (e.g. asymptotes, jumps) or infinite slope points, and may fail to graph the entire function.

This is caused due to the nature of the iterative formula which is used to approximate the function's behaviour: $$y_{n} = y_{n-1} + hf(x_{n-1}, y_{n-1})$$

Euler's method follows the slope of the function, which means that at the asymptote/infinite slope point, it is impossible to continue the generation of points.

For example, attemping to graph the equation \(\frac{dy}{dx}=-\frac{1}{x^2}\) with initial point \((1,1)\) will yield an incomplete graph, with the curve not going past \(x=0\).

Furthermore, the accuracy of Euler's method increases as the step size decreases. This is because there are more, shorter tangents to model the behaviour with, allowing behaviour to be modelled more accurately. Some functions may not be graphed accurately with larger step sizes, such as rapidly oscillating functions, or functions with steep slopes near the initial value.

In addition, if the function is concave up, Euler's method will approximate points which are below the actual curve, while if the function is concave down, the estimated points will be above the curve. This becomes more apparent as points further away from the given initial value are approximated.

Example Problem 2

Justin has been pulling a lot of all nighters recently. His sleep schedule can be modelled by the function \(s(t)= \frac{t(t+1)}{t^2 + 1}\), where \(s(t)\) represents how sleepy he feels at time \(t, t \ge 0\).

Justin wants to know how sleepy he felt on average during \(1 \le t \le 5\), given that he felt exactly \(1\) unit of sleepiness at \(t=0\). Unfortunately, he's too sleepy to be able to do math right now so you'll have to help him!

Answer

The answer is equal to \(\frac{1}{5-1}\int_{1}^{5} s(t) dt\), as we are finding the sum of sleepiness from \(t=1\) to \(t=5\), and dividing by \(4\) to find the average.

In order to calculate this integral using the Fundamental Theorem of Calculus, we can use the program to perform Euler's method. Using \(0.1\) as the step value, we find that the approximation found at \(t=1\) is \(1.510758\), and the approximation found at \(t=5\) is \(6.197107\).

Therefore, the final answer is \(\frac{1}{5-1}(6.197107 - 1.510758)=1.17158725\) units of sleepiness on average.
